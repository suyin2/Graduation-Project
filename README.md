# 基于 Streamlit 的论文知识库与技术情报分析系统

## 项目简介

本项目是一个面向学术论文检索与技术情报分析的毕业设计系统，采用 **数据库优先** 的处理思路，结合 **论文抓取、数据清洗、分类标注、去重入库与分析展示** 等功能，构建了一个可持续扩展的论文知识库后端，并通过 **Streamlit** 提供可视化交互界面。

系统支持用户输入研究主题后，优先从本地数据库中查询相关论文；当数据库内容不足时，再自动抓取 arXiv 数据进行补全，并将新数据清洗、分类、去重后写入数据库，最终返回查询结果与分析内容。

---

## 项目目标

本项目旨在实现一个具备以下能力的论文分析系统：

- 支持自然语言输入研究主题
- 自动解析查询参数
- 优先从数据库中检索论文数据
- 当数据不足时自动联网抓取 arXiv 论文
- 对抓取结果进行清洗、分类、标签标注与去重入库
- 支持后续的技术情报分析与报告生成
- 通过 Streamlit 提供简洁直观的前端展示界面

---

## 主要功能

### 1. 自然语言查询
用户可以直接输入研究主题或需求，系统自动提取关键词及相关参数。

### 2. 参数解析与查询改写
系统能够对用户输入进行解析，并生成更适合数据库查询或论文抓取的检索条件。

### 3. 数据库优先检索
优先从本地数据库中查询已有论文数据，提升响应效率，减少重复抓取。

### 4. arXiv 自动补全
当数据库中的论文数量不足时，系统自动调用 arXiv 接口抓取缺失数据。

### 5. 数据清洗与标准化
对抓取到的论文信息进行字段整理、格式规范与内容清洗。

### 6. 分类与标签管理
系统会对论文进行分类与标签标注，便于后续检索、统计与分析。

### 7. 去重入库
对新抓取论文进行去重判断，避免重复存储，提高数据库质量。

### 8. 分析与展示
将查询结果和分析内容通过 Streamlit 页面展示，方便用户查看与使用。

---

## 系统流程

整体流程如下：

用户输入主题  
→ 参数解析  
→ 查询改写  
→ 数据库检索  
→ 若数据不足则抓取 arXiv  
→ 数据清洗  
→ 分类与标签标注  
→ 去重入库  
→ 更新数据库记录  
→ 返回结果并展示分析内容
